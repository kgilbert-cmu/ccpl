Start the program with
    "python XML.py"
    or double-click the `.py` file

Commands to quit command-line tool:
	"quit"
	"q"

Commands to go to next page of listings:
	"next"
	" " (empty string, as in - type nothing and hit 'Enter')

Command to go back one page of listings:
    "back"

Type the number in brackets next to a title to open that listing in the browser.
